# Documentation Plugin

这个 plugin 聚焦文档生成和维护，把 commands、agents 和 templates 组合起来，用于自动生成、同步和校验项目文档。

## Included Commands

- `Generate API Documentation`
- `Generate README`
- `Sync Documentation`
- `Validate Documentation`

## Included Agents

- `api-documenter`
- `code-commentator`
- `example-generator`

## Included Templates

- API endpoint 模板
- Function docs 模板
- ADR 模板
